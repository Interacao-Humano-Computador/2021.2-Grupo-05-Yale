# 1.Introdução.

O objetivo das metas de usabilidade é definir quais são os fatores de qualidade de uso que devem ser priorizados no projeto, como serão avaliados ao longo do processo de design, e quais faixas de valores são inaceitáveis, aceitáveis ou ideais para cada indicador de interesse. Com frequência, essa priorização se baseia nos indicadores atuais de desempenho dos usuários ao utilizarem o sistema. Usabilidade refere-se a garantir que produtos interativos são fáceis de aprender, efetivas de usar e agrádavel pela perspectiva do usuário.
Envolve otimizar as interações que as pessoas têm com produtos interativos para permitir que eles realizem suas atividades no trabalho, na escola e no seu dia a dia (Sharp, Rogers, Preece, 2019).

## 2.1 Efetividade

<p style="text-align: justify;">
O sistema é efetivo quando realiza as atividades para as quais ele foi criado, ou seja, se o sistema faz o que deveria fazer é eficiente.

Um sistema eficaz permite que o usuário alcance os objetivos desejados durante o uso, realize suas tarefas, acesse as informações necessárias e o aprendizado do sistema é garantido.

Um exemplo de não efetividade é quando o sistema operacional começa uma atualização, mas não consegue finalizar.

</p>

## 2.2 Eficiência

<p style="text-align: justify;">
O sistema é eficiente quando permite efetividade (ou seja, faz o que deveria) por meio da realização de poucos passos pelo usuário
  
Um exemplo de ineficiência são menus com os objetivos confusos. Nesse caso, o usuário deve procurar em muitos tópicos o que deseja aumentando o tempo de uso.
</p>
  
## 2.3 Segurança 
<p style="text-align: justify;">
O sistema deve ser pensado na segurança, isso significa que o mesmo deve prevenir erros e reduzir riscos referente a ações drásticas, como deletar informações ou desligar o sistema.

Mas é importante ressaltar que para manter a segurança não é necessário limitar o poder de interação do usuário, pois isso infringe a regra de liberdade e autonomia do usuário.

</p>

## 2.4 Utilidade

<p style="text-align: justify;">
Cada sistema tem uma utilidade. Para fazê-la acontecer da melhor maneira é necessário fornecer ao usuário um conjunto de funções, estas que são esperadas pelo usuário naquele devido contexto.

Um bom exemplo é um filtro ou uma barra de busca

</p>

## 2.5 Aprendizagem

<p style="text-align: justify;">
É a capacidade de aprendizado do usuário ao utilizar o sistema. A meta é ser fácil de aprender a usar o sistema ou produto.
Se um sistema é muito difícil de usar, o usuário simplesmente desiste dele.
</p>

## 2.6 Memorização

<p style="text-align: justify;">
Esta é uma meta muito importante para sistemas que são usados eventualmente, pois demonstra a capacidade de memorização sobre a utilização do sistema.
O usuário não deve ser obrigado a reaprender a usar o sistema toda vez que o acessa.
</p>


# 3. As metas utilizadas e suas implicações no projeto
<p style="text-align: justify;">

- Eficácia: O sistema possui uma boa eficácia, pelo fato de não possuir muitas telas e funções, tornando os objetivos propostos acessíveis.
  
- Eficiência: O nível de produtividade após o aprendizado das tarefas será alto.
  
- Segurança: Os possíveis erros que podem ocorrer durante o manuseio do sistema são claramente especificados e retornam ao usuário a causa do erro, fazendo com que o mesmo possa corrigir sem auxílio de terceiros.
  
- Utilidade: Todas as funcionalidades do sistema são acessíveis sem dificuldade.
  
- Aprendizagem: O usuário pode aprender todas as funcionalidades de forma simples e fácil, caso possua o mínimo de conhecimento em navegação web, poderá aprender sem auxílio de terceiros.
  
- Memorização: Um usuário que não visita o sistema com frequência, terá facilidade em reutilizar os sistema.
</p>

# 4. Conclusão

<p style="text-align: justify;">
Com o estudo dessas metas, foi possível agrega-las ao projeto, tornando o uso do sistema simples e acessível ao usuário, evitando erros e gargalos para novos usuários e usuários que retornaram a utilizar o sistema.
</p>
